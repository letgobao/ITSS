# Getting Started
## Prerequisites
- npm
## Installation
1. Clone the repo
```
git clone https://github.com/letgobao/ITSS.git
```
2. Install NPM packages
```
npm i
```
3. Run
```
npm start
```

## UI

1. Login screen
```
http://localhost:3000/login
```
2. Manager screen
```
http://localhost:3000/
```
3. Profile screen
```
http://localhost:3000/profile
```
4. Achievement screen
```
http://localhost:3000/achievement
```
